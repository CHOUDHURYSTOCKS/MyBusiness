🧠 BIASES — QUICK REVISION POSTER

Cognitive Biases (Thinking Errors)

  • Confirmation bias → see what you believe
  • Anchoring bias → first info sticks
  • Availability bias → easy memories feel common
  • Overconfidence bias → overestimating ability

Social Biases (People Judgment)

  • Stereotyping → group assumptions
  • In-group bias → favor “us”
  • Halo effect → one trait affects all
  • Attribution bias → blame others, excuse self

Memory Biases (Distorted Recall)

  • Hindsight bias → “I knew it!”
  • Recall bias → selective memory
  • False memory → remembering things that didn’t happen

Decision Biases (Bad Choices)

  • Loss aversion → loss hurts more
  • Framing effect → wording changes choice
  • Sunk cost fallacy → past investment traps you

Research Biases (Distorted Data)

  • Sampling bias → wrong sample = wrong result
  • Observer bias → expectations affect results
  • Reporting bias → only selective results shown
